Kolproxy

Kolproxy is a custom interface for adventuring in the Kingdom of Loathing by adding additional useful functionality on top of the normal interface. These enhancements are especially useful for ascending, and an important design feature for kolproxy is that it will not accidentally hurt your run (even when there are bugs!).
